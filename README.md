# CNN Classification

1. buat python env
2. aktifkan dir env. contoh:
   -> .\CNN-venv\Scripts\activate (Windows) |
   -> source CNN-venv/bin/activate (Linux) |
   -> deactivate (nonaktif)
4. install package yang dibutuhkan:
   -> pip install -r requirements.txt
5. running program.
   -> python main.py
